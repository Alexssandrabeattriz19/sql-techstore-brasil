Projeto SQL desenvolvido para praticar modelagem de banco de dados e consultas utilizando SQL Server.

# TechStore Brasil - Projeto SQL

## Sobre o projeto

Este projeto foi desenvolvido durante meus estudos em SQL com foco na transição para a área de Análise de Dados.

O objetivo foi criar um banco de dados para uma loja fictícia de tecnologia e realizar consultas para responder perguntas de negócio.

## Tecnologias

- SQL Server
- SQL Server Management Studio (SSMS)
- GitHub

## Conceitos aplicados

- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE
- AND
- BETWEEN
- IN

## Estrutura do banco

Tabela: Produtos

Campos:

- ID
- Produto
- Categoria
- Marca
- Preço
- Estoque
- Cidade

## Exemplos de consultas

- Produtos acima de R$2.000
- Produtos da Dell
- Produtos entre R$500 e R$1.500
- Produtos das marcas Dell, Samsung e Logitech
- Produtos com estoque superior a 20 unidades

## Objetivo

Praticar conceitos fundamentais de SQL e desenvolver projetos para meu portfólio como Analista de Dados.
